About

What is Sentiment Analysis?

Sentiment analysis is the process of analyzing digital text to determine if the emotional tone of the message is positive, negative, or neutral.

About this Web Application

This web application is designed to automatically preprocess a CSV file of a dataset of collected online news articles. In this project, the dataset used is focused on English and Malay articles of Sabah environmental news.

Significance of this Project

Environmental news impacts public awareness and policy decisions. By understanding the tone of media coverage, organizations and individuals can respond more effectively to emerging issues.

Features

Key features of this web application model:

  • Upload a CSV file for analysis
  • Receive a summary of dataset results
  • Receive predicted sentiment
  • Receive data visualisations including charts and word clouds
  • Support for both English and Malay content
  • Simple and user-friendly interface
